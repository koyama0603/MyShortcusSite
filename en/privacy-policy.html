Privacy Policy for My Shortcuts

Last updated: June 22, 2026

My Shortcuts is a browser extension that provides a desktop-style workspace for web shortcuts, folders, sticky notes, media players, user-created mini-apps, and calendar reminders.

Data Stored Locally

My Shortcuts stores workspace data in the user's browser using IndexedDB. This can include:

  • Shortcut names, URLs, icons, and playback positions
  • Folder names and layout
  • Sticky-note contents and formatting
  • Calendar titles, notes, and dates
  • User-created or imported HTML mini-app source
  • Language, theme, window, zoom, minimap, and Picture-in-Picture settings

Browser and Website Data

When the user invokes a related feature, My Shortcuts may process:

  • Open tab titles, URLs, and favicons
  • A page or link URL selected through a context menu
  • Page titles, icons, and supported media metadata
  • Links dragged from supported pages or embedded windows
  • Supported YouTube or Spotify content loaded from those services
  • Direct files explicitly opened by the user

This processing is used only to provide the requested feature.

Data Transmission

My Shortcuts does not send workspace data, notes, calendar data, or browsing information to the developer's servers. Network requests go directly from the browser to the websites or media services needed for user-requested titles, icons, embeds, files, or searches.

User-Created HTML Mini-Apps

The extension's own JavaScript and CSS are bundled with the extension package. The optional HTML mini-app feature can run HTML and scripts supplied or imported by the user. External scripts referenced by that user-provided HTML run only inside a manifest-declared sandboxed environment without Chrome extension API access.

Third-Party Services

Embedded or opened content can be provided by services such as YouTube, Spotify, Vimeo, and web search providers. Their own terms and privacy policies apply to content loaded from those services.

Data Sharing, Advertising, and Analytics

My Shortcuts does not sell or rent user data. It does not use user data for advertising, tracking, analytics, or creditworthiness. The extension is currently ad-free.

Data Deletion

Users can delete individual items, empty the trash, remove the extension, or clear the extension's site data in Chrome. Removing extension data deletes locally stored workspace information.

Limited Use

Information handled by My Shortcuts is used only to provide the visible shortcut-management, workspace, media, calendar, and mini-app features described in the Chrome Web Store listing.
